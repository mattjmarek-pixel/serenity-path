## 2. Finalize app.json for store submission

Ensure ALL of the following fields are present and correctly set in `app.json`:
- name: "Serenity Path"
- slug: "serenity-path"
- version: "1.0.0"
- orientation: "portrait"
- icon: "./assets/images/icon.png"
- userInterfaceStyle: "automatic"
- splash: image "./assets/images/splash.png", resizeMode "contain", backgroundColor "#7B3FF2"
- ios.supportsTablet: false
- ios.bundleIdentifier: "com.serenitypath.app"
- ios.buildNumber: "1"
- ios.infoPlist.NSLocationWhenInUseUsageDescription: "Serenity Path uses your location to find nearby AA/NA meetings."
- ios.infoPlist.NSUserNotificationsUsageDescription: "Serenity Path sends daily reminders for reflections, check-ins, and gratitude practice."
- android.package: "com.serenitypath.app"
- android.versionCode: 1
- android.permissions: ["ACCESS_FINE_LOCATION", "ACCESS_COARSE_LOCATION", "RECEIVE_BOOT_COMPLETED", "VIBRATE"]
- extra.eas.projectId: "REPLACE_WITH_EAS_PROJECT_ID"

Leave all REPLACE_WITH_ values as-is for manual entry.

Serenity Path is a calm, private companion for your AA and NA recovery journey.
